# eyeZ

HTTP proxy experiment powered by [bubbletea](https://github.com/charmbracelet) and (httransform)[github.com/9seconds/httransform]

## Features

* Terminal UI
* Database recording (similar to [hyperfox](https://github.com/malfunkt/hyperfox))
